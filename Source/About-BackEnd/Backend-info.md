**About Git**
git clone + 깃헙 사이트에서 주소를 카피해 와서 붙여넣기 하면 된다.

rm -rf .git - 깃을 떨어뜨리고 독립된 소스가 된다.

npm init : 내가 노드 프로젝트를 시작한다고 알리는 것.

이걸 만들면 package.json 파일이 만들어진다.

name 을 물어보면 넣거나 넘어가면 된다.

app.js 파일을 만들고 시작하면 이 파일을 기본 서버 파일로 설정된다.
